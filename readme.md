# 🚗 DealerPulse Dashboard

A real-time dealership performance dashboard built to help CEOs and branch managers track sales, identify bottlenecks, and take action.

---

## 🌐 Live Demo

👉 https://your-vercel-link.vercel.app

---

## 📊 Features

* 📈 **Overview Dashboard**

  * Total Leads
  * Sales (Delivered)
  * Revenue
  * Conversion Rate

* 🏢 **Branch Filtering**

  * View performance per branch

* 📉 **Sales Chart**

  * Visual representation of sales by branch

* 🚨 **Actionable Insights**

  * Detects leads not contacted in 7+ days
  * Highlights potential revenue risks

* 🎯 **Target vs Achieved**

  * Tracks performance against sales targets
  * Helps identify underperforming branches

---

## 🧠 Key Insight

Example:

> "Hyderabad branch has multiple leads not contacted in 7+ days — high risk of drop-off."
