# DealerPulse Dashboard

## What I built
- Overview dashboard with KPIs
- Branch filter
- Sales chart
- Actionable insights (stale leads detection)
- Target vs Achieved tracking

## Why
The goal was to help dealership leadership quickly understand performance and act on bottlenecks.

## Key Decisions
- Focused on actionable insights instead of too many charts
- Simple and clean UI for clarity
- Client-side data processing for speed

## Tradeoffs
- No backend API
- Limited dataset simulation

## What I’d build next
- Forecasting (predict target achievement)
- Rep-level performance
- AI-generated summaries

## Insights Found
- Some leads are not contacted for over 7 days
- Certain branches are behind target
